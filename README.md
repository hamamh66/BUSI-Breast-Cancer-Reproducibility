# BUSI Breast Cancer Reproducibility Package

## Overview
This repository provides a reproducibility framework for auditing and visualizing reported results of a hybrid deep learning model for breast cancer classification using the BUSI ultrasound dataset.

## Features
- Predefined input data (CSV/JSON)
- Reproducibility notebook (Colab-ready)
- Consistency audit of reported metrics
- Automated generation of tables and figures
- Outputs saved to Google Drive

## Usage
1. Upload or clone this repository in Google Drive or Colab.
2. Open the notebook in Google Colab.
3. Run all cells sequentially.
4. Outputs will be saved to your Google Drive.

## Note
This repository operates in results-audit mode and does not perform new training or cross-validation.
